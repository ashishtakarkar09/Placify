# Placify 🚀

## Overview

Placify is an all-in-one placement preparation platform designed for engineering students. The application helps students track academic progress, monitor placement readiness, prepare for technical interviews, manage coding practice, and improve their chances of securing job opportunities.

The goal of Placify is to provide a centralized platform where students can access all essential placement preparation tools in one place.

---

## Features

### 📊 CGPA Predictor

* Predict required SGPA to achieve a target CGPA.
* Set academic goals for upcoming semesters.
* Visual progress tracking.

### 💻 DSA Tracker

* Track Data Structures and Algorithms preparation.
* Monitor completed and pending topics.
* Measure overall learning progress.

### 🧠 Aptitude Tracker

* Track quantitative aptitude preparation.
* Monitor reasoning and verbal practice.
* Analyze preparation progress.

### 🏢 Company Eligibility Checker

* Check eligibility based on CGPA and academic criteria.
* View company requirements.
* Identify placement opportunities.

### 🎤 Interview Preparation

* HR interview questions.
* Technical interview preparation.
* Subject-wise interview resources.

### 📄 Resume Builder

* Create and manage resumes.
* Organize projects, skills, and achievements.
* Improve placement readiness.

### 📝 Notes Management

* Store placement-related notes.
* Organize learning materials.
* Quick access to important information.
