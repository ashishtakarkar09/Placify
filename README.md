# Placify 🚀

## AI-Assisted Placement Preparation Platform

Placify is an AI-assisted placement preparation and career readiness platform designed specifically for engineering students. The application provides a centralized solution for tracking academic progress, preparing for placements, monitoring coding practice, analyzing company eligibility, and improving overall career readiness.

The platform combines multiple placement-focused tools into a single dashboard, helping students stay organized and focused throughout their placement journey.

---

## 🎯 Problem Statement

Engineering students often use multiple applications for placement preparation, including CGPA calculators, coding trackers, aptitude platforms, resume builders, and interview preparation resources.

Placify solves this problem by bringing all essential placement preparation tools together in one platform, providing a seamless and structured preparation experience.

---

## ✨ Features

### 📊 CGPA Predictor

* Calculate required SGPA for future semesters.
* Set academic targets.
* Track progress toward desired CGPA.

### 💻 DSA Progress Tracker

* Monitor Data Structures and Algorithms preparation.
* Track completed and pending topics.
* Measure overall learning progress.

### 🧠 Aptitude Preparation Tracker

* Track quantitative aptitude progress.
* Monitor logical reasoning and verbal preparation.
* Analyze strengths and weaknesses.

### 🏢 Company Eligibility Checker

* Check placement eligibility based on CGPA and academic criteria.
* Analyze company requirements.
* Identify suitable placement opportunities.

### 🎤 Interview Preparation

* Technical interview preparation.
* HR interview question bank.
* Subject-wise interview resources.

### 📄 Resume Builder

* Manage resumes and profile information.
* Organize projects, skills, and achievements.
* Improve placement readiness.

### 📝 Notes Management

* Create and organize placement-related notes.
* Store preparation resources.
* Quick access to important information.

### 📈 Placement Readiness Dashboard

* Track preparation progress.
* Monitor overall placement readiness.
* Visual analytics and performance insights.
